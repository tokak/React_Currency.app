## Döviz Kuru API Entegrasyonu

React kullanarak geliştirilmiş, anlık döviz kurlarını kullanarak para birimi dönüştürme işlemi yapan bir web uygulamasıdır. Uygulama, güncel verileri çekmek için Free Currency API servisini kullanmaktadır.

### Özellikler
- **Anlık Veri:** Döviz kurlarını API üzerinden dinamik olarak çeker.

### Kullanılan Teknolojiler
- Frontend: React (Vite)
- Veri Çekme: Axios
- İkonlar: React Icons
- API: Free Currency API


## Kurulum ve Çalıştırma

### 1 Projeyi klonlayın
git clone  https://github.com/tokak/React_Currency.app.git

### 2 Paketleri yükleyin 
cd currency_app
npm install

### 3 API Anahtarını Ayarlayın
src/components/Currency.jsx dosyası içerisindeki api keyinizi yazınız.

https://app.freecurrencyapi.com/ sitesinden api keyinizi alabilirsiniz.

### 4 Uygulamayı başlatın

npm run dev
